# ML / AI Portfolio

## Objectives

This portfolio showcases my work in **Machine Learning (ML)** and **Generative AI (GenAI)**, demonstrating practical experience in supervised learning, large language models, and Retrieval-Augmented Generation systems.

The projects highlight end-to-end implementation, from data preprocessing and model training to retrieval-based AI architectures.

## Learning Goals

- Use supervised learning methods for predictive modeling and business decision-making.
- Understand deep learning models, including large language models (LLMs).
- Design and build Retrieval-Augmented Generation (RAG) systems for document-based AI solutions.
- Explore best practices in data preprocessing, embeddings, vector search, prompt engineering, and hallucination evaluation.
- Prepare for production-level AI workflows, including MLOps practices.

## Portfolio Structure

- **Classification/**  
  Contains a supervised machine learning project focused on customer churn prediction, including model comparison and evaluation of classification approaches.

- **RAG/**  
  Contains a Retrieval-Augmented Generation (RAG) project that retrieves information from technical PDF documents and generates grounded answers using large language models.
